# Privacy Notice for the GitHub Pages Site

This repository contains the source code for a GitHub Pages website.

This privacy notice applies to the use of the published website on GitHub Pages, not to downloading, cloning, or viewing this repository on GitHub.

The website itself is designed to run JavaScript locally in the visitor's browser. No personal information is collected, stored, or otherwise processed by the site owner through the website itself.

However, the website is hosted through GitHub Pages. When visiting the published site, GitHub may process technical and usage data under GitHub's own privacy statement, such as server logs or other website-related data.

For information about GitHub's own data handling, see GitHub's Privacy Statement.
